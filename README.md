### About The Project
#### Socket
Socket是讓開發者可以在抽象層面和外部系統溝通的一個工具，開發者可以不用
去煩惱電腦笨路由器笩是如何讀取協定、等IP變成電子訊號傳遞到目的地。

#### 橢圓曲線密碼學
橢圓曲線密碼學是一種基於橢圓曲線數學的非對稱性加密演算法。

####  TCP 
TCP是一個可靠的傳輸協定，他是將所傳送的資料送達目的地，且資料本身是正確無誤，所以在資料傳出去後，必須由接收端確認無誤後，回應給傳送端，才能達到TCP的可靠性。
